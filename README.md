## Dependencies

- Node.js
- Express
- EJS
- Bcrypt
- BcryptJS
- Body-parser
- Cookie-session
- Cookie-parser
- Morgan 
- Eslint

## Dev Dependencies

- Chai
- Mocha
- Nodemon

## Getting Started

- Install all dependencies (using the `npm install` command).
- Run the development web server using the `node express_server.js` command.
